# 🧩 Frontend

## ⚛️ React
Es una librería de JavaScript para construir interfaces de usuario basada en componentes reutilizables. Utiliza un Virtual DOM para optimizar el renderizado y permite manejar el estado de la aplicación mediante hooks como `useState` y `useEffect`.

Se utiliza principalmente en aplicaciones SPA donde se necesita una UI dinámica, reutilizable y mantenible.

SPA: es una aplicación web que carga una única página y actualiza dinámicamente su contenido sin recargar, mejorando la experiencia del usuario.”

**👉 Respuesta:**
> “React es una librería para construir interfaces declarativas basadas en componentes. Utiliza un Virtual DOM para optimizar el renderizado y hooks para manejar estado y efectos. Lo uso principalmente en aplicaciones SPA donde necesito reutilización de lógica y una UI dinámica.”

---

## 🔷 TypeScript
Es un superset de JavaScript que agrega tipado estático. Permite detectar errores en tiempo de desarrollo, mejorar la mantenibilidad y escalar proyectos grandes con mayor seguridad.

Es especialmente útil en equipos y aplicaciones complejas donde el tipado reduce bugs y mejora la autocompletación.

**👉 Respuesta:**
> “TypeScript extiende JavaScript con tipado estático, permitiendo detectar errores en desarrollo y mejorar la mantenibilidad. Lo uso especialmente en proyectos grandes donde la escalabilidad y la claridad del código son importantes.”

---

## ⚡ Vite
Es una herramienta de build y desarrollo frontend que utiliza ES Modules para ofrecer un entorno de desarrollo extremadamente rápido con Hot Module Replacement (HMR). Para producción, genera builds optimizados usando Rollup.

Se destaca por su velocidad frente a herramientas tradicionales como Webpack.

“ES Modules es el sistema estándar de módulos en JavaScript que permite importar y exportar código entre archivos, facilitando la organización y optimización de aplicaciones modernas.”

“HMR es una técnica que permite actualizar módulos de una aplicación en tiempo real sin recargar la página, mejorando la experiencia de desarrollo y manteniendo el estado de la aplicación.”

**👉 Respuesta:**
> “Vite es un bundler moderno que usa ES Modules para ofrecer recarga instantánea en desarrollo. Lo prefiero frente a Webpack en proyectos nuevos por su rapidez y simplicidad, y porque genera builds optimizados para producción.”

---

## 🎨 Tailwind CSS
Es un framework de CSS utility-first que permite diseñar directamente en el HTML mediante clases predefinidas, evitando escribir CSS personalizado.

Es ideal para mantener consistencia visual y acelerar el desarrollo de interfaces.

**👉 Respuesta:**
> “Tailwind es un framework utility-first que permite construir interfaces rápidamente usando clases reutilizables. Lo uso para mantener consistencia en el diseño y reducir la necesidad de escribir CSS personalizado.”

---

## 🧭 React Router
Es una librería de enrutamiento para React que permite manejar navegación entre vistas sin recargar la página, característica clave en aplicaciones SPA.

Permite definir rutas dinámicas, layouts y protección de rutas.

**👉 Respuesta:**
> “React Router permite manejar rutas en aplicaciones React, facilitando la navegación sin recargar la página. Lo uso para estructurar vistas y manejar rutas dinámicas en aplicaciones SPA.”

---

# 🖥️ Backend

## 🟢 Node.js
Es un entorno de ejecución de JavaScript en el servidor basado en el motor V8. Utiliza un modelo asincrónico y no bloqueante, ideal para manejar múltiples conexiones concurrentes.

Se utiliza en APIs, microservicios y aplicaciones en tiempo real.

**👉 Respuesta:**
> “Node.js permite ejecutar JavaScript en el backend con un modelo asincrónico y no bloqueante. Es ideal para aplicaciones escalables y con alta concurrencia, como APIs o servicios en tiempo real.”

---

## 🚀 Express
Es un framework minimalista para Node.js que facilita la creación de servidores y APIs mediante middleware.

Tiene un ecosistema muy amplio y es fácil de aprender.

**👉 Respuesta:**
> “Express es un framework ligero para Node.js que simplifica la creación de APIs mediante middleware. Lo uso cuando necesito desarrollar rápido y aprovechar su gran ecosistema.”

---

## ⚡ Fastify
Es un framework para Node.js enfocado en alto rendimiento. Incluye validación de esquemas, serialización eficiente y mejor manejo de performance que Express.

Es ideal para aplicaciones que requieren velocidad y escalabilidad.

**👉 Respuesta:**
> “Fastify es un framework web para Node.js enfocado en rendimiento. A diferencia de Express, ofrece validación y serialización nativa. Lo elegiría en aplicaciones donde la performance y la escalabilidad son críticas.”

---

## ⚖️ Express vs Fastify
- **Express** → más simple, mayor comunidad, desarrollo rápido  
- **Fastify** → mejor rendimiento, validación integrada  

**👉 Respuesta:**
> “Express es más simple y tiene mayor ecosistema, mientras que Fastify ofrece mejor rendimiento y validación integrada. Elegiría Express para prototipos o desarrollo rápido, y Fastify para aplicaciones de alto tráfico.”

---

## ✅ Zod
Es una librería de validación de esquemas para TypeScript que permite definir estructuras de datos y validar en runtime, generando además tipos automáticamente.

Ayuda a mantener consistencia entre backend y frontend.

**👉 Respuesta:**
> “Zod permite definir esquemas para validar datos y generar tipos automáticamente en TypeScript. Lo uso para asegurar consistencia entre lo que recibo en runtime y en código.”

# 🐍 Backend

## 🐍 Python
Es un lenguaje de alto nivel, conocido por su simplicidad y legibilidad. Se utiliza ampliamente en backend, ciencia de datos e inteligencia artificial.

Tiene un ecosistema muy grande y versátil.

**👉 Respuesta:**
> “Python es un lenguaje versátil y legible, muy usado en backend y ciencia de datos. Lo elijo cuando busco productividad y un ecosistema amplio.”

---

## ⚡ FastAPI
Es un framework moderno para construir APIs en Python, basado en type hints. Ofrece alto rendimiento, validación automática y documentación interactiva (Swagger).

Es comparable en rendimiento con Node.js.

“Type hints son anotaciones de tipo en Python que permiten indicar qué tipo de datos se espera en variables y funciones. No se validan en runtime por defecto, pero mejoran la legibilidad, el tooling y permiten validaciones automáticas en frameworks como FastAPI.”

**👉 Respuesta:**
> “FastAPI es un framework para construir APIs rápidas en Python que usa type hints para validación automática. Lo uso cuando necesito alto rendimiento con bajo esfuerzo en documentación y validación.”


---

## 🌐 Uvicorn
Es un servidor ASGI liviano y de alto rendimiento que permite ejecutar aplicaciones modernas de Python como FastAPI.

Maneja múltiples conexiones de forma asincrónica.

“ASGI (Asynchronous Server Gateway Interface) es una interfaz estándar para aplicaciones Python que permite comunicación asíncrona con el servidor, soportando WebSockets y alta concurrencia.”

**👉 Respuesta:**
> “Uvicorn es un servidor ASGI que permite ejecutar aplicaciones Python asincrónicas como FastAPI. Se destaca por su rendimiento y manejo eficiente de múltiples conexiones.”
